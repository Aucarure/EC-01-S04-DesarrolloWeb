# 🛒 DevPanel — Gestión de Productos
 
Aplicación web para administrar un inventario de productos. Permite crear, editar y eliminar productos con una interfaz moderna y oscura.
 
---
 
## 🚀 Requisitos previos
 
Antes de empezar, asegúrate de tener instalado en tu máquina:
 
- [Node.js](https://nodejs.org/) — versión 16 o superior
- [Git](https://git-scm.com/)
 
Puedes verificar si ya los tienes con:
 
```bash
node -v
git -v
```
 
---
 
## 📦 Instalación y ejecución
 
Sigue estos pasos en orden:
 
### 3. Instala las dependencias
 
```bash
npm install
```
 
### 4. Inicia el servidor
 
```bash
node app.js
```
 
### 5. Abre el navegador
 
Ve a [http://localhost:3000](http://localhost:3000)
